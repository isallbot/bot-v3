# BOT ISALL V3
Simple WhatsApp Bot

### FOR TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
> pkg install imagemagick -y
> git clone https://github.com/isallbot/bot-v3
> cd bot-v3
> npm i
> node index.js
```

---------

### FOR WINDOWS/VPS/RDP USER
```bash
> git clone https://github.com/isallbot/bot-v3
> cd bot-v3
> npm i
```
###### Run
```bash
> node index.js
```
